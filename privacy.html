Privacy Policy

Last updated: July 22, 2025

This application is used privately by Yaiquel Carmenates and does not collect, store, or share personal data from users. All case status checks are made directly to the official USCIS API using read-only queries.

Data Handling

No user data is stored or processed by this application. Case numbers are queried temporarily and discarded after retrieval.

Cookies & Tracking

This website does not use cookies or third-party tracking tools.

Third-Party APIs

The application connects to the official USCIS Case Status API for status updates. No modification or interference is made to that system.
