Privacy Policy

Last updated: July 23, 2025

This Privacy Policy explains how data is handled when using the USCIS Case Tracker application developed by Yaiquel Carmenates.

1. Types of Data Collected

No personal data is collected, stored, or shared. This includes:

  • Geolocation data
  • Financial information
  • Medical data
  • User contacts
  • Identifiers or cookies

2. Data Usage

All queries are sent directly to the public USCIS API. Case numbers are used temporarily and discarded immediately after response. No tracking, storage, or profiling occurs.

3. Third Parties and Data Sharing

No data is shared with any third parties. No analytics, advertising, or tracking integrations are used.

4. Data Sale and Monetization

User data is not sold or monetized in any form.

5. User Rights and Control

Users have full control and transparency. They may stop using the app at any time without any effect on data (since none is stored).

6. Breach Notification

In the extremely unlikely event of a data breach, users will be notified via email immediately.

7. Data Retention Policy

No data is stored. Each interaction is stateless.

8. Data Deletion

Since no data is retained, there is nothing to delete. Should future changes require storage, users will be given deletion options.

9. Transfer of Ownership

If ownership changes, users will be notified. No data will be transferred without consent.

10. Data Disposal

If temporary logs are generated, they are securely discarded using encryption and proper protocols.

11. Account Management

No user accounts are created. There is no account to manage or close.

12. Changes to This Policy

Any future changes will be clearly posted here, with date and explanation. Users will be notified of major updates.

13. Consent

Consent is not required under the current design. If future versions collect any data, consent will be explicitly required.
